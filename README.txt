Musson Stove Calculator

Описание:
Калькулятор подбора пиролизной печи Муссон для отопления помещений с учетом теплопотерь, типа древесины, времени горения и рабочего режима.

Функции:
- Расчет теплопотерь здания с учетом материала стен, площади окон и дверей, утепления крыши.
- Сравнение моделей печей Муссон по мощности и энергии одной закладки.
- Расчет расхода дров в зависимости от времени работы печи и характеристик древесины.
- Возможность выбора породы древесины, цены за м³ и заполнения топки.
- Визуализация результатов в виде таблицы и графиков.

Запуск:
1. Установите Python и Streamlit.
2. Скачайте app.py и README.txt в одну папку.
3. В командной строке перейдите в эту папку и выполните:
   ```
   streamlit run app.py
   ```

Файлы:
- app.py — основной файл приложения Streamlit.
- README.txt — инструкция по использованию приложения.

Примечание:
Расчеты являются ориентировочными. Для точного подбора оборудования обратитесь к специалистам.
